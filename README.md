Este é um projeto de Jogo da Velha desenvolvido com HTML, CSS e JavaScript puro.

O objetivo do projeto foi praticar:

Manipulação do DOM

Eventos de clique

Controle de estado do jogo

Alternância de jogadores (X e O)

Verificação automática de vitória e empate

Organização de código em funções

O jogo sorteia automaticamente quem começa e permite reiniciar a partida a qualquer momento.
